# Notification sounds

These locally bundled MP3 files are converted from Mixkit WAV sound effects:

- `info.mp3`: [Confirmation tone](https://mixkit.co/free-sound-effects/tones/), item 2867
- `warning.mp3`: [Double beep tone alert](https://mixkit.co/free-sound-effects/alerts/), item 2868
- `critical.mp3`: [Orchestral emergency alarm](https://mixkit.co/free-sound-effects/alerts/), item 2974

Mixkit publishes these sound effects under its [Sound Effects Free License](https://mixkit.co/license/). Mixkit states that its sound effects may be used in personal and commercial projects without attribution. This file is retained to document asset provenance.
